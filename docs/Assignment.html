Web Terminology;

  • The Internet ;It is a massive network of networks. It connects millions of computers together globally, forming a network in which any computer can communicate with any other computer as long as they are both connected to the Internet.
  • The World Wide Web ;It is a way of accessing information over the medium of the Internet. It is an information-sharing model that is built on top of the Internet (A bunch of connected computers that have websites).
  • Browser ;It is a program that allows you to interact and view contents of the Web (Application programs dedicated for web browsing e.g. opera, Firefox, chrome, Internet Explorer, etc. are called browsers).
  • client ;It a devices (e.g. laptop, tablet, mobile phones etc.) from which browsers are used.
  • server ;It is a dedicated machine which are there to serve the client,it is always running 24/7
    *DNS Server ;It contains all the addresses of all the servers,it stores IP Address .

What is a Web Page?

  • Web pages are text files containing HTML
  • HTML – Hyper Text Markup Language
    • A notation for describing
  • document structure (semantic markup)
  • formatting (presentation markup)
  • HTML permits the platform- independent display of information across network.
  • The markup tags provide information about the page content structure

Creating HTML Pages

  • HTML files can be created with text editors:
    • NotePad, WordPad,NotePad ++, PSPad
  • Or HTML editors (WYSIWYG Editors):
    • Microsoft FrontPage
    Macromedia Dreamweaver
    Netscape Composer
    Microsoft Word
  • An HTML file must be saved with a .htm or .html file extension

HTML Structure

  • HTML is comprised of “elements” or “tags”
  • Elements (tags) are nested one inside another: Begins with < html > and ends with < /html >
  • HTML describes structure using two main sections: < head > and < body >

HTML Code Formatting

  • The HTML source code should be formatted to increase readability and facilitate debugging.
    Every block element should start on a new line.
    Every nested (block) element should be indented.
    Browsers ignore multiple whitespaces in the page source, so formatting is harmless.
  • For performance reasons, formatting can be sacrificed
